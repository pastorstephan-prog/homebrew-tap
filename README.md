# Homebrew tap for AgentLaunch Doctor

Install the stable release:

```bash
brew install pastorstephan-prog/tap/agentlaunch-doctor
agentlaunch-doctor --all-user-agents
```

AgentLaunch Doctor is a read-only health and privacy diagnostic CLI for macOS LaunchAgents. The formula builds from the tagged source release locally with Swift; it does not install a background service.

Project documentation and security policy: https://github.com/pastorstephan-prog/agentlaunch-doctor
